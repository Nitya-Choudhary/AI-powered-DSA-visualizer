# AI-Powered DSA Visualizer

An interactive web app to visualize Data Structures and Algorithms with step-by-step animations and AI-based explanations.

## Features
- Visualize sorting algorithms (Bubble Sort, etc.)
- Step-by-step execution
- Play, pause, and reset controls
- Clean and interactive UI
- AI-based explanation (future scope)

## Tech Stack
- React.js
- JavaScript
- Tailwind CSS
